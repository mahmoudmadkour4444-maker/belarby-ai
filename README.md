# بالعربي AI 🚀

مساعد دراسي ذكي بالعربية — صنعه يوسف عماد

## رفع على Vercel (بدون كمبيوتر)

### الطريقة السهلة — Vercel مباشرة:

1. روح [github.com](https://github.com) وأنشئ حساب
2. أنشئ Repository جديد اسمه `balarabi-ai`
3. ارفع كل ملفات هذا الزيب على الـ Repository
4. روح [vercel.com](https://vercel.com)
5. اضغط "Sign Up" وادخل بـ GitHub
6. اضغط "New Project"
7. اختار الـ Repository بتاعك
8. اضغط "Deploy" ✅

الموقع هيشتغل على رابط مجاني مثل:
`https://balarabi-ai.vercel.app`

## تشغيل محلي

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
